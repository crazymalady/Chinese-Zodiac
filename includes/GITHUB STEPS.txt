1. Initialize the git 
   > git init .
2. Add the files you want to add
   > git add .
3. You are on master branch already
   > git status
4. Commit the files you want to add
   > git commit -m "Added Files"
5. Copy the URL in repo and Add Remote Origin
   > git remote add origin https://...
6. Check if origin is already there
   > git remote -v
7. Add upstream
   > git remote add upstream httpsss.
